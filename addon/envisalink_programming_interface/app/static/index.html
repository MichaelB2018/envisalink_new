Welcome — Panel Configuration Sync

This tool will read all programmable settings from your Vista 20P panel — zone types, zone names, system options, reporting, custom words, and keypad configuration — so you can view and edit them from your browser.

⚠️ Before you begin — please read carefully:
  • The panel will enter programming mode during the sync. While in programming mode, zones are not monitored — the alarm will not respond to faults or trigger alerts.
  • If a power outage, network interruption, or software crash occurs during the sync, the panel may remain stuck in programming mode. You will need to press *99 on a physical keypad to exit manually.
  • You should have a physical keypad accessible at all times so you can recover from an interrupted session.
  • The panel must be disarmed before syncing.

⏱ The initial sync takes approximately 3–6 minutes. You will see real-time progress on the LCD display above.
